# Planning Poker - Full project (backend + simple frontend)

This project is a full-stack Planning Poker prototype:
- Backend: PHP, file-based JSON storage
- Frontend: Bootstrap + vanilla JS (fetch)

Run locally:
1. From project root:
   /Applications/MAMP/bin/php/php8.2.*/bin/php -S localhost:8000 -t public
2. Open http://localhost:8000 in your browser (the frontend is served from /public)
